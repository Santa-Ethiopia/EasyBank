This project is designed to practice Spring Boot features in a realistic context by developing a simplified banking system. The application allows users to register themselves, after which an administrator can approve or reject their registration. Customers can manage bank accounts, perform transactions such as deposits, withdrawals, and view their transaction history. In addition to direct customer transactions, the system introduces the concept of agents, who act as intermediaries authorized to perform deposits and withdrawals on behalf of customers. The system also includes an admin dashboard for user management and monitoring activities. 
The backend is built with Spring Boot, leveraging Spring Security for authentication and role-based access control, and MySQL for transactional data storage. The front-end interface is developed using Thymeleaf, providing a simple but functional dashboard for both customers and administrators.
This project is not intended for production use as a secure financial platform, but rather serves as a practice environment to explore and apply Spring Boot features, database integration, semantic web technologies, and basic web security in a banking domain.
